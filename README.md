## About The Project

[![Future Screen Shot][product-screenshot]](https://example.com)

Lycans will be a 3D strategy game using vocal proxy. It is currently In-Dev by Pagaloma Studio.

Planned Features:

- Fist Person View
- Several roles
- Tasks
- Day / Night cycle
- Voting system
